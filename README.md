# 📋 Agente Inteligente de Tareas con Claude

## Descripción

Un asistente inteligente alimentado por la IA de Claude que gestiona tu agenda de tareas de forma natural e intuitiva. Combina el poder del procesamiento de lenguaje natural con herramientas de visualización moderna.

## Características principales

**Interfaz Dual:**
- 🖥️ **CLI Interactivo**: Comunícate con el agente en lenguaje natural
- 🌐 **Dashboard Web**: Visualiza tus tareas con gráficos dinámicos

**Funcionalidades:**
- ✨ Crea tareas simplemente escribiendo lo que necesitas
- 📊 Visualiza progreso con gráficos de Matplotlib
- 💾 Persistencia automática en archivo JSON
- 🎯 Gestiona prioridades (Alta, Media, Baja)
- 📈 Estadísticas en tiempo real

**Tecnología:**
- **Backend**: Python + Anthropic API (Claude 3.5 Sonnet)
- **Frontend**: Flask + HTML/CSS/JavaScript
- **Visualización**: Matplotlib para gráficos
- **Almacenamiento**: JSON para persistencia

El agente comprende comandos naturales como "crea una tarea urgente", "¿cuántas tareas me faltan?" o "marca todo como completado", automatizando tu gestión de tareas de manera inteligente.

**Perfecto para:** Productividad personal, organización de proyectos y seguimiento de actividades.
